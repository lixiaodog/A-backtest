# Backtrader A股回测系统 - 前后端扩展计划

## 技术栈

### 后端（Python）
- **Web框架**: Flask (Python 轻量级 Web 框架)
- **回测引擎**: 复用现有 backtrader 系统
- **实时通信**: Flask-SocketIO (WebSocket)
- **任务队列**: 线程池（简化版）

### 前端（JavaScript）
- **UI框架**: React + Ant Design
- **图表库**: Lightweight Charts (TradingView 开源版)
- **通信**: Socket.IO 客户端

## 系统架构

```
┌─────────────┐     HTTP/WS      ┌─────────────┐
│   Browser   │ ◄──────────────► │   Flask     │
│   (React)   │                  │   Backend   │
└─────────────┘                  └──────┬──────┘
                                         │
                                    ┌────▼────┐
                                    │ Backtrader│
                                    │  Engine  │
                                    └──────────┘
```

## 实现步骤

### 1. 后端搭建（Flask + Python）
- 创建 `backend/` 目录
- 创建 `app.py` - Flask + SocketIO 主应用
- 依赖：`flask`, `flask-socketio`, `python-socketio`, `eventlet`
- 实现 API：
  - `POST /api/backtest` - 提交回测任务
  - `GET /api/backtest/<task_id>` - 获取回测结果
- WebSocket 推送回测进度

### 2. 前端搭建（React + JavaScript）
- 创建 `frontend/` 目录
- 使用 Vite 初始化 React 项目
- 依赖：`antd`, `lightweight-charts`, `socket.io-client`

### 3. 前端组件
- `BacktestForm` - 参数表单（股票代码、日期、策略）
- `TradeViewChart` - K线图（TradingView 风格）
- `ProgressPanel` - 实时进度、资金曲线

### 4. 实时功能
- WebSocket 连接
- 回测进度实时推送
- K线图动态标记交易信号
