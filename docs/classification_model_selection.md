# 分类训练模型选择指南

## 📊 系统支持的分类模型

当前系统支持以下分类模型：

### 1. RandomForest（随机森林）

**优点**：
- ✅ 鲁棒性强，不易过拟合
- ✅ 对特征尺度不敏感（可以不归一化）
- ✅ 能处理高维数据
- ✅ 能评估特征重要性
- ✅ 训练速度快
- ✅ 支持多分类
- ✅ 对缺失值不敏感

**缺点**：
- ⚠️ 模型较大，预测速度较慢
- ⚠️ 对噪声敏感
- ⚠️ 不适合处理线性关系

**适用场景**：
- 特征数量多（220个因子）
- 数据量中等（几千到几万样本）
- 需要稳定的预测结果
- 不确定特征之间的关系

**推荐指数**: ⭐⭐⭐⭐⭐

---

### 2. LightGBM

**优点**：
- ✅ 训练速度快
- ✅ 预测速度快
- ✅ 内存占用小
- ✅ 准确率高
- ✅ 支持GPU加速
- ✅ 能处理大规模数据

**缺点**：
- ⚠️ 容易过拟合（需要调参）
- ⚠️ 对特征尺度敏感（需要归一化）
- ⚠️ 参数较多，调参复杂
- ⚠️ 对噪声敏感

**适用场景**：
- 数据量大（几万到几百万样本）
- 需要快速训练和预测
- 有GPU资源
- 愿意花时间调参

**推荐指数**: ⭐⭐⭐⭐

---

## 🎯 分类任务推荐

### 推荐方案：RandomForest

**理由**：

1. **适合当前数据规模**
   - 特征数：220个因子
   - 样本数：几千到几万条
   - RandomForest在这个规模下表现优秀

2. **稳定性好**
   - 不易过拟合
   - 对参数不敏感
   - 默认参数就能有不错的效果

3. **不需要额外处理**
   - 对特征尺度不敏感
   - 即使不归一化也能正常工作
   - 对缺失值不敏感

4. **可解释性强**
   - 可以查看特征重要性
   - 了解哪些因子对预测贡献大

5. **支持多分类**
   - 适合3分类（买入/持有/卖出）
   - 适合5分类（强烈买入/轻度买入/持有/轻度卖出/强烈卖出）

## 💡 使用建议

### 1. 首选：RandomForest

```python
# 推荐配置
model_type = 'RandomForest'
mode = 'classification'
n_estimators = 100  # 树的数量
max_depth = 10      # 树的最大深度
```

**适合**：
- 初次训练
- 不确定最佳参数
- 需要稳定的结果
- 不想花太多时间调参

### 2. 备选：LightGBM

```python
# 推荐配置
model_type = 'LightGBM'
mode = 'classification'
n_estimators = 100
max_depth = 10
learning_rate = 0.1
num_leaves = 31
```

**适合**：
- 数据量大（>10万样本）
- 有GPU资源
- 愿意花时间调参
- 追求更高的准确率

### 3. 集成模型（最佳）

```python
# 同时训练多个模型，然后融合预测
models = ['RandomForest', 'LightGBM']
```

**优点**：
- 结合多个模型的优势
- 提高预测稳定性
- 降低过拟合风险

## 🚀 实际应用建议

### 对于分类任务：

1. **第一步：使用RandomForest**
   - 快速建立baseline
   - 了解模型性能
   - 查看特征重要性

2. **第二步：尝试LightGBM**
   - 如果RandomForest效果不理想
   - 尝试调参优化
   - 对比两个模型的效果

3. **第三步：集成学习**
   - 同时使用RandomForest和LightGBM
   - 融合预测结果
   - 提高稳定性

### 参数建议：

**RandomForest**：
```python
n_estimators = 100      # 树的数量，越多越好但会慢
max_depth = 10          # 树的深度，防止过拟合
min_samples_split = 5   # 分裂所需最小样本数
min_samples_leaf = 2    # 叶节点最小样本数
```

**LightGBM**：
```python
n_estimators = 100      # 树的数量
max_depth = 10          # 树的深度
learning_rate = 0.1     # 学习率
num_leaves = 31         # 叶子节点数
min_child_samples = 20  # 叶节点最小样本数
```
